# Usage

- `npm install`
- `apt-get install libasound2`
- `python3 -m venv .venv`
- `source .venv/bin/activate`
- `pip install -r requirements.txt`
- `cd plugins && pip install -e . && cd ..`  # Install multilingual plugin
- `mkdocs serve`

# Components and Features
- Framework: [MkDocs](https://www.mkdocs.org)
- Custom Theme: Allows subpages for pages
- RSS Feed: The blog entries are in a RSS feed
- English and German. If not manually translated then it is automatic on build time
